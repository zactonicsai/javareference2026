// dual-write architecture CQRS-lite
Writes (POST/PUT/DELETE) start a Temporal workflow on the product-task-queue. The workflow runs in the spring-demo-worker container and calls an activity that persists to Postgres. Reads (GET) bypass Temporal and query Postgres directly. The Temporal Web UI is at http://localhost:8088.
